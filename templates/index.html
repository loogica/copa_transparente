O projeto Copa Transparente Cidadã

O Copa Transparente é um projeto sem fins lucrativos, sem nenhum tipo de associação com partidos ou organizações políticas, projetado para manter atualizados os dados do portal Copa Transparente do Governo.

Problema

O Governo disponibiliza os dados em um formato pouco amigável para extração de informações cruzadas. Somar valores, relacionar colunas e entidades(Execuções Financeiras, Licitações, Empreendimentos e outras) se torna muito trabalhoso e sucetível a erros, se feito de forma semi-automatizada ou manual.

Solução

Foi criado um software que faz o download de um arquivo de uma URL pública do governo e gera esse site para que os dados estejam sempre atualizados e trabalhados pro usuário final, seja ele um cidadão, como eu, ou um profissional.

Código Livre

Você pode obter o código do programa que faz o download e gera o site aqui: https://github.com/loogica/copa_transparente
